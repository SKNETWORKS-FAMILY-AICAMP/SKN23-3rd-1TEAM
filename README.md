# 📄 프로젝트 개요 (Overview)

## ✦ 프로젝트명

> **이력서 기반 맞춤형 AI 역량 검증 및 압박 면접 플랫폼**

<br>

## ✦ 프로젝트 기간

> 2026.02.27(금) ~ 2026.03.03(화) **(5일)**

<br>

## ✦ 프로젝트 소개

사용자의 이력서(PDF)를 분석하여 직무와 경험에 기반한 맞춤형 면접을 제공하는
초개인화 AI 면접 솔루션입니다.

단순한 1문 1답식 챗봇을 넘어, 지원자의 답변을 실시간으로 분석하고
모순점이나 깊이 있는 기술적 지식을 파고드는
LangGraph 기반 압박 꼬리 질문 시스템을 구현했습니다.

또한 WebRTC(웹캠 미러링) 및 STT(음성 인식)를 통해
실제 화상 면접과 동일한 긴장감과 UI/UX를 제공합니다.

<br>

## ✦ 프로젝트 필요성 (배경)

### ✶ 정형화된 면접 대비의 한계

* 시중 면접 서비스는 뻔한 질문 반복에 그침
* 실제 지원자의 역량 검증에 한계 존재

### ✶ 맞춤형 모의 면접의 부재

* 프로젝트 경험 및 기술 스택 기반 심층 질문 환경 필요

### ✶ 효율적인 채용 검증 도구 필요

* 대량 이력서 검토 및 1차 스크리닝에 시간/비용 과다 소모

<br><br>

# 💼 비즈니스 이해 (Business Understanding)

## ✶ B2C (구직자 대상 에듀테크 SaaS)

* 이력서 기반 무한 반복 AI 모의 면접 제공
* **상위 % 상대평가 대시보드(백분위)** 제공
* 문항별 피드백 및 레이더 차트 시각화
* 게이미피케이션 기반 성장 지표 제공

## ✶ B2B (기업용 HR 채용 솔루션)

* 면접관 주관적 편향 최소화
* DB 설계, 알고리즘, CS 지식 등 역량 수치화
* 1차 서류/면접 스크리닝 리소스 절감


<br><br>

# 📄 프로젝트 목표

### ✶ 동적 RAG (Continuous Learning RAG) 구축

* 이력서 데이터 + 실시간 답변을 Vector DB(ChromaDB)에 즉시 Append
* 이전 답변 맥락을 기억하는 AI 면접관 구현

### ✶ LangGraph 기반 AI 제어 로직

* 답변 채점 → 꼬리 질문 생성 → 주제 전환
* 논리적 분기 처리 시스템 구현

### ✶ Dual DB 아키텍처 설계

* **RDBMS (SQLite)** : 질문 및 메타데이터 관리
* **Vector DB (ChromaDB)** : AI 검색 및 문맥 유지
* 투트랙 구조 분리 및 동기화

### ✶ 상용 서비스급 프론트엔드 (Streamlit)

* WebRTC 카메라 미러링
* STT 마이크 연동
* 로그인/설정 모달
* 실시간 타이머
* 카카오톡 스타일 UI
* 결과 분석 대시보드 구현

<br>

# ✨ 주요 기능 (Key Features)

1. **📄 이력서 기반 맞춤형 질문 생성 (PDF Parsing & RAG)**
   - 지원자의 이력서를 업로드하면 PDF를 분석하여 프로젝트 경험, 기술 스택을 파악하고 그에 맞는 고도화된 면접 질문을 자동 생성합니다.
2. **🕸️ LangGraph 기반 꼬리 질문 & 압박 면접 시스템**
   - 단순한 문답형 챗봇이 아닙니다. 지원자의 답변 논리를 분석하여 헛점이 있거나 깊은 지식이 필요한 경우 논리적인 꼬리 질문(Follow-up Question)을 던집니다.
3. **🎙️ WebRTC & STT 기반 실전 면접 환경**
   - 웹캠 미러링과 음성 인식(STT) 기능을 통해 실제 화상 면접과 동일한 긴장감과 UI/UX를 제공합니다. 타이머 기능으로 실전 감각을 극대화합니다.
4. **📊 상세 결과 대시보드 및 백분위 평가**
   - 면접 종료 후, 답변에 대한 상세 피드백(모범 답안, 개선점)을 제공하며, 가상 DB를 활용하여 전체 지원자 중 나의 상위 % (백분위) 및 역량 레이더 차트를 시각화합니다.

# 📄 프로젝트 내용

## ✦ 데이터셋 (Dataset)

AI의 초기 **Cold Start 문제를 방지**하고
Dual DB를 세팅하기 위해 자체 제작 데이터셋을 활용합니다.



### ✶ 직무별 면접 질문 셋 (questions.csv)

* 파이썬 백엔드 개발자 맞춤형 핵심 질문 30선

<br>

### 구조

* `question_id`
* `job_role`
* `difficulty`
* `question_text`
* `expected_keywords`

### 활용

* FastAPI 구동 시

  * SQLite (메타데이터 DB)
  * ChromaDB (Vector DB)
    양쪽에 동시 주입

<br>

### ✶ 테스트 가상 이력서 (resumes/)

* pdfplumber 기반 전처리 파이프라인 테스트용
* 표/다단 형식 포함 Python 백엔드 지원자 이력서 PDF 4종

<br>

### ✶ 상대평가(백분위) 산출용 가상 DB

* 평균 75점 / 표준편차 10 기반 정규분포 생성
* 가상 지원자 점수 500건 Seed 데이터 생성
* 모의 면접 종료 후 상위 % 계산에 활용
